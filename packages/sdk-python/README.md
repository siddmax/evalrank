# EvalRank Python SDK

Python SDK package boundary for public EvalRank APIs.

The current SDK surface re-exports public contracts and fixtures from `evalrank-core`, including `CapabilityFingerprintInput`, `RawEntry`, `EvaluationRequest`, `CandidateSet`, `StageCandidate`, `EvidenceItem`, `ResultRow`, `EvidenceSet`, `Exclusion`, and `TheCall`.
