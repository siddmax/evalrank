# EvalRank Python SDK

Python SDK package boundary for public EvalRank APIs.

The current SDK surface re-exports public contracts and fixtures from `evalrank-core`, including `CapabilityFingerprintInput`, `RawEntry`, `EntityRef`, `EvaluationRequest`, `CandidateSet`, `StageCandidate`, `EvidenceItem`, `EvidenceSet`, `ResultRow`, `UseCaseCatalog`, `RankingGroup`, `Exclusion`, `TheCall`, `RankedEntity`, `Recommendation`, `PUBLIC_FIXTURE_KINDS`, and `sample_public_fixture`.
